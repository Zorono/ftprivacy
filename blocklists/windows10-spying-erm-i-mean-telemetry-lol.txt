# Microsoft Windows 10 and Xbox One DNS blocking lists by ftpihole
# https://www.ftpihole.com/

# Blocks, to the best of my ability, ads and "telemetry" from Windows 10
# Designed for and tested on the Pi Hole but should work on all DNS based filters

# I have intentionally left update domains out of this so Windows Update will work correctly. This is for security reasons.

# If any domain here damages functionality on your computer please pin down the domain(s) via the Pi Hole web UI and open up an issue on GitHub, it will be looked into, but please do give enough detail so we can replicate the issue, be certain we're removing the correct domain(s), and be aware of what privacy protection we're losing as a result.

# As a general rule, we will try to weigh these issues up. For example if a domain being blocked stops MS Edge from working, but other browsers work fine, we would encourage you to use another browser or whitelist the domain on your network. If, on the other hand, the domain is confirmed to stop internet connections from working correctly - clearly we will urgently remove it. You get the idea.
