# General computer (Windows, MacOS, Linux) DNS blocking list by ftpihole - https://www.ftpihole.com/

# I frequently watch the traffic from my Windows 10 laptop after each major update to add to the list if needed.
# You will want to use both however as the Windows list only blocks Microsoft related services while this one blocks ads and trackers from all over the net.

# The Windows "telemetry" list can be found here: https://ftpihole.s3.eu-west-2.amazonaws.com/blocklists/windows10-spying-erm-i-mean-telemetry-lol.txt

# Designed for and tested on the Pi Hole but should work on all DNS based filters including HOSTS blocking software running locally on your machine.
# However I have not tested it on any such software so I cannot promise compatibility.
# I recommend using a Pi Hole because it will block unwanted domains across every device on your entire network!

# As the name suggests this file is and will be focused on blocking ads, analytics, location guessing scripts, and other nasties. If you use Windows 10 I highly recommend you add the Windows 10 specific list as well as this one. See link just a couple paragraphs above.

# The aim with this list is to keep things working correctly while still reducing ads and trackers as much as possible.

# Further "risky" lists that block more but could break things, or lists that are explicitly designed to block certain sites (looking your way Zuck), are on ftpihole.com with more coming soon if you are feeling wild.
